Commit 1 · Inline SVG diagrams

Editorial diagrams gallery

Five figures on a single page — architecture, flowchart, sequence, quadrant, funnel — each rendered with the rules paraphrased from diagram-design (MIT). Editorial-Diagram aesthetic: Instrument Serif + Geist + Geist Mono, rust accent.

  • Shape carries meaning: oval / rect / diamond / merge-dot in the flowchart
  • Arrows drawn before nodes — nodes sit on top
  • Every arrow label has a paper-colored masking rect behind it
  • Bottom horizontal legend strip, never floating
  • ≤ 2 accent uses per figure; one italic-serif annotation callout with dashed leader
  • Funnel widths honest to the conversion numbers

Commit 2 · Hyperframes long-form

Long-form explainer composition

A 60-second 16:9 explainer on "How our cache saved $40k/month." Five scenes with eyebrow + headline fade-ups, a $52,000 → $12,000 count-up pair, and a three-node progressive SVG diagram reveal. Works as a live HTML preview (auto-plays) and as a Hyperframes composition (`window.__timelines["longform-demo"]`, paused and ready for the engine).

  • On page load, the timeline plays — you should see scene 1 fade in, stagger, then dwell before scene 2
  • No console errors (open DevTools and confirm)
  • Diagram reveal in scene 3: nodes fade-scale in, then arrows stroke-dasharray in
  • Count-up animates smoothly from 0 to the target number
  • GSAP timeline is paused in Hyperframes capture mode (gated by `window.__hyperframesCapture`)

Commit 2 · Hyperframes reel · 9:16 vertical

Brain-rot reel — phone / Shorts / Reels

A 32-second vertical reel on "Partition by tenant, not by type." Seven beats (HOOK → PROBLEM → CONTEXT → MECHANISM → PROOF → RESOLUTION → CTA), hard cuts every 1.2–1.8s, word-by-word kinetic typography, two-stage progressive diagram reveal, burned-in captions in the phone safe-zone.

  • HOOK (0–2s): 4,200 counts up; "requests per second" label lands
  • PROBLEM (2–5s): "Your queue chokes at 500." word-stagger, "chokes" in accent
  • MECHANISM (9–18s): two SVG stages — shared queue → per-tenant lanes
  • PROOF: 8.4× scale-in with back-ease; "throughput in prod" label
  • Captions fade in at each beat, out at the end, all readable in the bottom third
  • No console errors; all GSAP constraints satisfied

Same reel, landscape canvas
