What Changed In Practice

Routing moved up

Command choice is no longer implied only by command prose. The repo now has a top-level resolver.

Formats became first-class

PDF, MP4, MOV, and WebM are explicit output surfaces instead of being treated as incidental flags.

Video got recipe routing

generate-video and render-video now sit above recipe-specific intent like reels, overlays, and browser loops.

Static drift now fails

Manifest/filesystem mismatch, missing resolver coverage, doc drift, unknown formats, and missing route coverage are all checked.

Representative Routes

“Export this deck as a PDF”
prompt -> output family `pdf` -> `export-pdf`

No need to hide the user’s real intent inside /generate-slides --pdf.

“Make a transparent lower third for Premiere”
prompt -> `render-video` -> `overlay-transparent` -> `mov`

Editor-targeted output is now distinct from default playback delivery.

“Turn this changelog into a 35s reel”
prompt -> `generate-video` -> `social-reel` -> `mp4`

Prompt shape and recipe routing are explicit.

“Extract keyframes from this draft video”
prompt -> output family `assets` -> `export-assets`

Asset export is a first-class command surface now.

“Compare this RFC to the codebase visually”
prompt -> `plan-review` -> `html`

Core HTML explainers stay where they belong.

“Make a browser-native transparent loop”
prompt -> `render-video` -> `browser-loop` -> `webm`

Browser playback is no longer conflated with editor delivery.
